# Analyse-des-crimes-avec-python
Ce projet analyse les crimes commis à San Francisco en 2016

Ce projet utilise les données de « kaggle » pour étudier les
phénomènes criminels dans la ville de San Francisco sur une année
en utilisant la technologie python.

Après Analyse, il en resulte que:

- le nombre de crimes signalés était de 150 430 ;
-le sud, le nord et le centre sont des nids d’infractions ;
- le vol constitue le délit le plus rependu ;
- le samedi et dimanche comptent plus de délits ;
-le mois de janvier est le mois où beaucoup de crimes sont commis ;
-la tendance de délits était en baisse ;
- le taux de résolution est 28.40 %
